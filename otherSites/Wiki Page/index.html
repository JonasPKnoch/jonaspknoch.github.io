Gérard Historical Fiction

by Jonas Knochelmann

This project is an attempt to create a realistic, but entirely fictional historical event and the intertwining web of connections surrounding it. To be clear, none of the people, things, or events described in this story really existed, but the goal was to seamlessly weave them into real history in a believable fashion.

With that in mind, the following pages:

  • Guillaume-Léon Gérard
  • Gérard Engine
  • Jean Frère
  • Deciphering of the Gérard Engine
  • Pascal Zagré
  • Aurélie Noe

Were written by me. However, they contain many links to other people, things and events that really did exist. So please, explore the narrative naturally and follow any links that sound interesting or you want to know more about, as they are meant to be part of the narrative as well.
